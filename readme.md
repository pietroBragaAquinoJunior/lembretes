A idéia desse projeto, é possibilitar o salvamento de lembretes.\
A informação não ficará salva em um banco de dados, mas no navegador.\
Escolhi nuxt.js pois é muito simples dar deploy na vercel por exemplo, por ser 100% javascript.\
O usuário poderá exportar a informação para não perdê-la.

Como usar usar localmente?

1. Clone o repositório.
2. Instale dependências
```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```
3. Rode o projeto localmente.
```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```
